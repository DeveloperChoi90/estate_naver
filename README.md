# Naver Estate Crawler (부동산 매물 통합 조회)

네이버 부동산의 매물 정보를 효율적으로 조회하고 관리할 수 있는 파이썬 기반 크롤링 및 트래킹 도구입니다.

## 🚀 주요 기능

- **지역 기반 검색**: 사용자가 입력한 지역(예: 마포구, 광명시 등)의 아파트 매물을 검색합니다.
- **다양한 거래 유형**: 매매, 전세, 월세 조건을 선택하여 조회할 수 있습니다.
- **맞춤형 정렬**: 랭킹순, 낮은 가격순, 높은 가격순, 최신 등록순 정렬 기능을 제공합니다.
- **상세 정보 분석**: 매물의 층, 방향, 면적뿐만 아니라 현관구조(계단식/복도식 등) 정보를 추가로 수집합니다.
- **관심 매물 저장**: 조회된 매물 중 관심 있는 항목을 선택하여 `selected_listings.csv` 파일에 저장(누적)할 수 있습니다.
- **지도 링크 제공**: 검색한 지역과 조건이 적용된 네이버 부동산 지도 링크를 자동으로 생성하여 제공합니다.

## 🛠 설치 방법

1. **Python 설치**: Python 3.8 이상의 버전이 필요합니다.
2. **라이브러리 설치**: 아래 명령어를 통해 필요한 패키지를 설치합니다.
   ```bash
   pip install pandas requests selenium
   ```
3. **Chrome 브라우저 및 드라이버**: Selenium 사용을 위해 최신 버전의 Chrome 브라우저가 설치되어 있어야 합니다. (ChromeDriverManager 등을 통해 드라이버가 자동 관리되거나 경로에 있어야 함)

## 💻 실행 방법

프로젝트 루트 디렉토리에서 아래 명령어를 실행합니다.

```bash
python main.py
```

### 실행 프로세스:
1. **지역 입력**: 조회하고 싶은 지역명을 입력합니다 (예: '마포구').
2. **거래 유형 선택**: 1(매매), 2(전세), 3(월세) 중 하나를 선택합니다.
3. **정렬 기준 선택**: 1(랭킹순), 2(낮은 가격순), 3(높은 가격순), 4(최신 등록순) 중 하나를 선택합니다.
4. **출력 개수 설정**: 화면에 표시할 매물의 개수를 입력합니다.
5. **결과 확인 및 저장**:
   - 조회된 리스트를 확인합니다.
   - 저장하고 싶은 매물의 번호를 입력하면 'selected_listings.csv' 파일에 저장됩니다.
   - 출력된 네이버 부동산 링크를 통해 웹브라우저에서 직접 상세 내용을 확인할 수 있습니다.

## 📁 파일 구조

- `main.py`: 프로그램의 메인 실행 파일 및 CLI 인터페이스
- `tracker.py`: 네이버 부동산 API 호출 및 데이터 가공 로직
- `api_client.py`: 네이버 부동산 API 통신을 위한 클라이언트 클래스
- `naver_estate_crawling.py`: Selenium 기반의 웹 크롤링 기능 (상세 정보 수집용)
- `utils.py`: 가격 포맷팅 등 공통 유틸리티 함수
